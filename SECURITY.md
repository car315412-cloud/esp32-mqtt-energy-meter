# Security Policy

If you find a security issue in this repository, please report it privately rather than opening a public issue.

## Please avoid publishing

- Wi-Fi SSID and password
- MQTT broker credentials
- API keys or service tokens
- Device certificates or private keys

## Recommended handling

- Replace secrets with placeholders in example code.
- Keep personal or school network settings outside the public repository.
- Review any screenshot or report image before publishing it.
